AquaMinds es una plataforma integral de código abierto diseñada para el monitoreo y análisis de la calidad del agua en tiempo real, integrándose con dispositivos IoT basados en ESP32.

Esta aplicación permite a investigadores, estudiantes y gestores ambientales visualizar datos críticos obtenidos de sensores remotos de forma centralizada y segura.

Características Principales:
• Monitoreo IoT: Conexión con medidores ESP32 para lectura de pH, turbidez, temperatura y conductividad.
• Visualización en Tiempo Real: Gráficas y datos actualizados al instante para la toma de decisiones.
• Gestión de Espacios de Trabajo: Organiza múltiples medidores y proyectos en entornos separados.
• Sistema de Alertas: Notificaciones inmediatas sobre estados críticos del agua.
• Reportes Profesionales: Generación de informes detallados en formato PDF para análisis y archivo.

Transparencia Técnica:
• Desarrollada en Flutter para un rendimiento nativo multiplataforma.
• Comunicación segura con una API backend basada en FastAPI.
• Licencia MIT: Software libre, sin publicidad y auditable por la comunidad.
